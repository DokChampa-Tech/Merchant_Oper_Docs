# การสร้างบัญชีเข้าใช้งานสำหรับ Oper และ การทำ Partial Match
เป็นการสอนวิธีสร้างรหัสผู้เข้าใช้งานสำหรับตำแหน่ง Oper ของร้านค้า และ การทำ Partial Match ด้วยสลิปการโอนเงินจากลูกค้า (กรณีรายการไม่ Success)

#* การสร้างรหัสผู้เข้าใช้งานสำหรับตำแหน่ง Oper
1. ทำการป้อนข้อมูลสำหรับเข้าสู่ระบบ Back Office ของร้านค้า.
2. จากนั้นให้กด "Sign In With Email And Password" เพื่อเข้าสู่ระบบ.

![1-2](/assets/login.png)

3. เลือก "จัดการผู้ใช้งาน" จากแถบเมนูด้านซ้าย.
4. กดที่ "ผู้ใช้งาน" เพื่อไปยังหน้ารายการผู้ใช้งาน.
5. กดที่ "NEW USER".

![5](/assets/newUserButton.png)
